## 🚀 Quick Start

```bash
# Clone repository
git clone https://github.com/ZAYNINFINITY/portfolio-web.git
cd portfolio-web

# Install dependencies
npm install

# Configure EmailJS (optional)
cp .env.example .env
# Add your EmailJS credentials

# Start development server
npm run dev
```

---

## 🔨 Build & Deploy

```bash
# Development
npm run dev

# Production build
npm run build

# Preview production build
npm run preview
```
